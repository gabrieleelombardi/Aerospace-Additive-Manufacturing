# Topology Optimization and Additive Manufacturing for Aerospace Applications

This repository presents a university project focused on the design, structural analysis, topology optimization, and additive manufacturing of aerospace components.

## Project overview

The work includes finite element analyses, topology optimization, PolyNURBS-based geometry refinement, material selection, additive manufacturing process evaluation, and preliminary cost estimation.

## Main activities

- Static and dynamic structural analysis
- Finite Element Method validation
- Topology optimization
- PolyNURBS geometry reconstruction
- Design of lightweight aerospace components
- Additive manufacturing process evaluation
- Material comparison between Ti6Al4V and Al2139
- Preliminary cost estimation

## Tools and methods

- Altair Inspire
- Altair Inspire Print 3D
- Bambu Studio
- FEM analysis
- Topology optimization
- Additive manufacturing
